# Internacionalização (i18n)

## Decisão

Usado o **i18next** diretamente sobre **Context API**, com locale extraído da URL.

```
/en-US/transactions → English
/pt-BR/transactions → Português
```

## Fluxo

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│    URL      │ ──→ │  Context    │ ──→ │  i18next    │
│  (/pt/...)  │     │  (locale)   │     │  (t())      │
└─────────────┘     └─────────────┘     └─────────────┘
```

## Estrutura de Keys

```json
{
  "transactions": {
    "title": "Transactions",
    "form": {
      "description": "Description",
      "amount": "Amount"
    }
  }
}
```

**Justificativa**: estrutura flexível ao tamanho da aplicação, pacotes pequenos e compatível com ferramentas de automação.

## Alternativa

Poderia usar cookie para persistir preferência do usuário independente da URL.
