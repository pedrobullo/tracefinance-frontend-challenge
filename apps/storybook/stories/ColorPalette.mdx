# Color Palette

Our design system uses a comprehensive color palette with support for both light and dark themes.
All colors are defined as CSS custom properties (tokens) and automatically adapt based on the active theme.

## Usage

### CSS Variables

```css
.my-element {
  background-color: var(--background-level-one);
  color: var(--text-primary);
  border-color: var(--border-primary);
}
```

### Tailwind CSS Classes

```jsx
<div className="bg-level-one text-primary border-border-primary">Content</div>
```

### Theme Switching

The theme automatically switches when the `.dark` class is applied to a parent element:

```jsx
<div className="dark">{/* All children will use dark theme colors */}</div>
```
